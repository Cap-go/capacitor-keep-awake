# Changelog

## 1.0.0 (Initial Release)

- Initial release
- Support for iOS, Android, and Web platforms
- `keepAwake()` - Prevent the device from dimming the screen
- `allowSleep()` - Allow the device to dim the screen
- `isSupported()` - Check if keep awake is supported
- `isKeptAwake()` - Check if the device is currently kept awake
- `getPluginVersion()` - Get the native plugin version
