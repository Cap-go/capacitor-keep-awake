# @capgo/keep-awake Example App

This is a simple example app demonstrating the @capgo/keep-awake Capacitor plugin.

## Getting Started

1. Install dependencies:
   ```bash
   bun install
   ```

2. Build the web app:
   ```bash
   bun run build
   ```

3. Add a platform:
   ```bash
   npx cap add ios
   # or
   npx cap add android
   ```

4. Sync the project:
   ```bash
   npx cap sync
   ```

5. Open in native IDE:
   ```bash
   npx cap open ios
   # or
   npx cap open android
   ```

## Development

Run the dev server:
```bash
bun run start
```
